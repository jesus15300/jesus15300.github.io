Sistema de tutorados BUAP

Sistema de Gestion de tutorados es un sistema encargado de mostrar a un usuario Tutor toda la informacion acerca de los alumnos que fueron asignados como sus tutorados.

El sistema fue creado con PHP puro, HTML, JavaScript en conjunto con Ajax para interactuar con el servidor.

El sistema incluye dos vistas, una para el tutor y una para el tutorado.

El tutor es capaz de agregar a sus tutorados asi como registrar a cada uno sus calificaciones en sus respectivas materias, tambien el tutor es capaz de visualizar la informacion de su carrera, su porcentaje, en que facultad estudia, y el mapa corricular del alumno de forma grafica.

El alumno es capaz de visualizar su informacion en tiempo real asi como las materias que esta cursando y las posibles del siguiente semestre tambien el alumno puede visualizar su mapa grafico que incluye calificaciones y materias cursadas.

Clone Band

Clone band es un "videojuego" en desarrollo el cual intenta ser una copia del famoso videojuego llamado Rock Band.

Este videojuego es desarrollado en el motor grafico Unity en conjunto con el lenguaje C#.

Actualmente el videojuego ya realiza una lectura de un archivo midi que puede estar ubicado en cualquier lugar del disco duro y proporcionar toda la informacion de los objetos que se debe de instanciar a lo largo de una cancion.

Otra de las funcionalidades que ya se tiene es la funcion de cargar todos las pistas de audio existentes en la ruta seleccionada para asi asignarlas a una mezcladora de audio de cada respectivo instrumento.

El videojuego tambien es capaz de calcular cada uno de los "Beats" del metronomo para asi guiar al jugador visual y auditivamente.

De momento solo se puede interactuar con el videojuego con un teclado para rasguear las notas y asi mostrar las animaciones que se tienen hechas, en lo que se desarrolla un sistema de imput para diversos perifericos.

MidiSplitter


MidiSplitter es un proyecto que realice cuando realizaba transmisiones en vivo tocando la bateria electrica, este proyecto fue creado en el lenguaje de c# usando la libreria TeVirtualMidi.

Esta herramienta surgio cuando necesitaba mi instrumento digital en varios programas de audio, pero no existe programa que me permita compartir mi bateria con dos o mas editores de audio y que en terminos de latencia sea inaudible, asi que cree este programa que me permite escuchar mi bateria en un editor de audio asi como jugar un juego con ella.

El programa obtiene los diversos dispositivos midi que tengo y despues me permite transmitirlo en los dispositivos virtuales que le nombre.
