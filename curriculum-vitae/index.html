Sistema de tutorados BUAP

Sistema de Gestión de tutorados es un sistema encargado de mostrar a un usuario Tutor toda la información acerca de los alumnos que fueron asignados como sus tutorados.

El sistema fue creado con PHP, HTML, JavaScript en conjunto con Ajax para interactuar con el servidor.

El sistema incluye dos vistas, una para el tutor y una para el tutorado.

El tutor es capaz de agregar a sus tutorados así como registrar a cada uno sus calificaciones en sus respectivas materias, también el tutor es capaz de visualizar la información de su carrera, su porcentaje, en qué facultad estudia, y el mapa curricular del alumno de forma gráfica.

El alumno es capaz de visualizar su información en tiempo real así como las materias que está cursando y las posibles del siguiente semestre. También el alumno puede visualizar su mapa gráfico que incluye calificaciones y materias cursadas.

Clone Band

Clone Band es un "videojuego" en desarrollo que intenta ser una copia del famoso videojuego llamado Rock Band.

Este videojuego se desarrolla en el motor gráfico Unity en conjunto con el lenguaje C#.

Actualmente, el videojuego ya puede leer archivos MIDI que pueden estar ubicados en cualquier lugar del disco duro y proporcionar toda la información sobre los objetos que se deben instanciar a lo largo de una canción.

Otra de las funcionalidades que ya se ha implementado es la capacidad de cargar todas las pistas de audio existentes en la ruta seleccionada para asignarlas a una mezcladora de audio de cada respectivo instrumento.

El videojuego también puede calcular cada uno de los "beats" del metrónomo para guiar al jugador visual y auditivamente.

De momento, solo se puede interactuar con el videojuego mediante un teclado para rasguear las notas y así mostrar las animaciones que se han creado, mientras se desarrolla un sistema de entrada para diversos periféricos.

Midi Splitter

Midi Splitter es un proyecto que realicé cuando hacía transmisiones en vivo tocando la batería eléctrica. Este proyecto fue creado en el lenguaje C# usando la librería TeVirtualMidi.

Esta herramienta surgió cuando necesitaba usar mi instrumento digital en varios programas de audio, pero no existía un programa que me permitiera compartir mi batería con dos o más editores de audio y que en términos de latencia fuera inaudible. Así que creé este programa que me permite escuchar mi batería en un editor de audio y, al mismo tiempo, jugar un juego con ella.

El programa detecta los diversos dispositivos MIDI que tengo y luego me permite transmitirlos en los dispositivos virtuales que le asigné.

Juego de los emojis

Inspirado en Suika Game, este videojuego fue creado para explorar el uso de pantallas táctiles en Unity. La dinámica principal involucra soltar emojis en un contenedor para agruparlos por tipo, permitiéndoles crecer y avanzar de nivel.

Desarrollé este juego de forma independiente en el motor Unity, desafiándome a interpretar la entrada táctil y convertirla en coordenadas de juego. Inicialmente, comienzas con un emoji gigante que actúa como el jugador; al tocar la pantalla, puedes cambiar la posición donde caerá el emoji más pequeño.

La mecánica central del juego se centra en la fusión: cuando un emoji cae sobre otro del mismo tipo, se fusionan y suben de nivel, ubicándose en la posición media entre ellos.

El sistema de puntuación se basa en multiplicar el nivel del emoji por un factor específico.

El juego concluye cuando el emoji más alto cruza una línea punteada roja, siendo la línea amarilla una advertencia indicativa de que el jugador está cerca de perder.

En un futuro, tengo contemplada la posibilidad de crearle un servicio web que le ayude a guardar las puntuaciones de los jugadores y así crear un sistema de ranking global.

Invernadero inteligente

Este programa se desarrolló como proyecto final para la materia de Integración de Sistemas. El proyecto consta de dos programas: uno creado en Arduino para la arquitectura del ESP32, encargado de monitorizar los sensores de una planta, y el segundo programa desarrollado en Windows Presentation Foundation (WPF), diseñado para visualizar los parámetros de la planta y modificar las condiciones ambientales.

El software de Arduino puede monitorizar el ambiente de la planta leyendo sensores de humedad y de temperatura para lograr calcular las condiciones optimas de la planta, al igual el software puede activar un foco para proporcionar calor a la planta cuando baje la temperatura, encender una bomba de agua para regarla y abrir una puerta para ventilar su entorno.

El software de WPF realiza peticiones a la placa Arduino a través del puerto serie, procesa la información para mostrarse de manera intuitiva y tiene la capacidad de enviar datos a la placa Arduino para configurar nuevos parámetros, como la temperatura mínima que la planta puede soportar o la humedad deseada, entre otros.
